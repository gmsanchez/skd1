# SKD-1
ROS 2 packages for the SKD-1, our experimental 4WD skid-steer vehicle. This project is built and tested on Ubuntu 24.04 and ROS 2 Jazzy Jalisco.

This is the main repo, containing the base packages for operation on the robot hardware. There are also four more repositories containing complementary material:
 
- [skd1_sensors](https://github.com/gmsanchez/skd1_sensors): Sensor packages for SKD-1.
- [skd1_visualization](https://github.com/gmsanchez/skd1_visualization): Visualization packages for SKD-1 using RViz.
- [skd1_simulation](https://github.com/gmsanchez/skd1_simulation): Simulation packages for SKD-1 using Gazebo.
- [skd1_extra](https://github.com/gmsanchez/skd1_extra): Extra packages, containing design files and BoM.
- [hibachi_firmware](https://github.com/gmsanchez/hibachi_firmware): Firmware for the STM32 Blue Pill low level controller.
